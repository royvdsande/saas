How the new FitFlow UI works

The frontend is refreshed, but the logic remains grounded in the same Firebase and Stripe systems.

1. Sign in

Use Google, email/password, or a magic link through Firebase Authentication. Account state updates the workspace immediately.

2. Upgrade

The premium modal creates a checkout session in Firestore. Stripe redirects back with success or cancel URL parameters.

3. Sync premium

Premium status is read from user data, Stripe-related Firestore records, and local browser state for anonymous purchases.

4. Link local purchases

If a user bought premium anonymously and later signs in, the app offers to connect that local purchase to their account.
