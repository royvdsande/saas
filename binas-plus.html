Premium tier

One premium purchase, same proven backend flow

The premium plan uses the existing Stripe price and Firestore checkout-session integration. Only the surface design changed.

💳

Stripe checkout

Premium checkout continues to create Firestore checkout session documents and redirect users into Stripe securely.

🔗

Account linking

Anonymous purchases can be linked later to a signed-in account, so premium follows the user across devices.

🛡

Admin control

Admins can still grant or revoke premium access from the admin dashboard without changing the security model.
